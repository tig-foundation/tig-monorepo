## Additional Notes

`there_v5` is a GPU-based exact vector search algorithm.

Main points:
- Exact search, not approximate
- Uses batching to handle large database scans efficiently
- Uses an optimized memory layout to improve GPU performance
- Built to improve runtime compared with earlier versions

Fuel note:
- Higher fuel may be needed on larger tracks
- In practice, this version may require up to 500b fuel across all tracks
