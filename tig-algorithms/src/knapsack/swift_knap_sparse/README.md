## Additional Notes

Sparse-track specialist derived from `swift_knap`.
Key changes: wider sparse-core exploration, higher DP refinement threshold,
more perturbation rounds on small and medium instances, and broader refinement
budget where sparse interaction structure can support deeper search.

Intended regime:
- sparse and medium-density knapsack tracks
- higher absolute solution quality where additional refinement is worth the fuel
