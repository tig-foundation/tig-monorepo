Dense-track specialist derived from `swift_knap`.
Key changes: tighter core window sizing on dense interaction matrices,
lower DP refinement threshold, fewer expensive hybrid merge attempts,
and stronger perturbation pressure for faster escape from dense local minima.

Intended regime:
- dense / high-interaction knapsack tracks
- quality-per-fuel optimization over absolute peak search depth
